# Model Context Protocol (MCP) Services

This directory contains MCP services for AI integrations and model context management.

## Structure

Each MCP service should follow the structure:
```
services/mcp/[service-name]/
├── package.json
├── src/
│   ├── index.ts
│   ├── handlers/
│   └── types/
├── Dockerfile
└── README.md
```
