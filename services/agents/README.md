# Agent Services

This directory contains autonomous agent services for various platform operations.

## Structure

Each agent service should follow the structure:
```
services/agents/[agent-name]/
├── package.json
├── src/
│   ├── index.ts
│   ├── agents/
│   └── utils/
├── Dockerfile
└── README.md
```
